# 十五五粮食平台数据治理课题

## 背景
几个省的粮食平台要做数据治理，目的是让客户有意愿升级现有平台。

## 计划讨论时间
下周一、二（具体日期待定）

## 讨论问题

### 1. AI如何融入粮食平台？
- 在流程里加？还是外围（如问答）？
- 哪些是痛点刚需？还是技术驱动？
- AI加入对省局会有效率提升吗？
- 哪些场景会提升效率？
- 哪些场景是客户觉得有需要的？

### 2. 数据基础问题
- 要做AI（如问答或其他），是否都需要先理数据？
- 四个省都没有完全用到数仓吧？

### 3. 其他驱动因素
- 除了AI驱动，还有其他驱动因素吗？
- 比如纯粹业务调整？
- 或者之前的一些bug？这种多吗？

### 4. 客户视角
- 如果你是省局客户，哪些能打动你，让你愿意投钱？

## 目标
集思广益，形成方案，推动客户升级现有平台。
