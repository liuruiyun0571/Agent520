# 部署验证清单

## 部署前检查

### 环境准备
- [ ] Docker 20.10+ 已安装
- [ ] Docker Compose 2.0+ 已安装
- [ ] 端口 80、3000、5432 未被占用
- [ ] 至少 2GB 内存可用
- [ ] 至少 10GB 磁盘空间

### 配置文件
- [ ] `.env` 文件已创建并配置
- [ ] `JWT_SECRET` 已设置为强密码
- [ ] 数据库密码已修改
- [ ] 钉钉 Webhook 配置（如需要）

## 部署步骤

1. **启动服务**
   ```bash
   docker-compose up -d
   ```

2. **验证服务状态**
   ```bash
   docker-compose ps
   ```
   - [ ] db 状态为 `healthy`
   - [ ] backend 状态为 `healthy`
   - [ ] frontend 状态为 `Up`

3. **初始化数据库**
   ```bash
   docker-compose exec backend node scripts/initDatabase.js
   ```

4. **验证 API**
   ```bash
   curl http://localhost:3000/health
   ```
   - [ ] 返回 `{"status":"ok"}`

5. **验证前端**
   - [ ] 访问 http://localhost 能打开登录页
   - [ ] 登录功能正常

1. `docker-compose logs` 查看服务日志

3. 检查 `.env` 配置是否正确
